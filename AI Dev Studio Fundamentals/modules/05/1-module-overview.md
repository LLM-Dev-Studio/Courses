# Module 5 Overview: The WinUI Desktop App

The WinUI desktop app provides a full-screen, native Windows experience for managing AI Dev Studio projects. Where the VS Code extension gives you four focused panels, the desktop app gives you 18+ pages covering everything from agent transcripts to codebase insights.

## What you will learn

- How to launch the WinUI app and navigate its main areas
- The key pages for day-to-day use: Agent Dashboard, Board, Decisions, Knowledge Base
- How to manage multiple projects and configure project settings
- The advanced views available only in the desktop app

## Lessons in this module

1. Launching the Desktop App
2. Key Pages: Agents, Board, and Decisions
3. Project and Knowledge Management
4. Advanced Views: Transcripts, Insights, and Codebase
5. Module Quiz

> [!IMPORTANT]
> The WinUI desktop app requires Windows 10.0.19041 or later and must be built with `-p:Platform=x64`. It is not available on macOS or Linux.

## Key Takeaway

The WinUI desktop app is the richest surface for project management, offering dedicated pages for every domain concept and visualisation tools not available in the VS Code extension.
