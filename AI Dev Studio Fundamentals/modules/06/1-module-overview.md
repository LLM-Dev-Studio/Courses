# Module 6 Overview: Multi-Agent Workflows

Now that you know how the platform works and how to use both UIs, this module takes you deeper into the workflows that make multi-agent orchestration effective — decision lifecycle, local LLM setup, and planning sessions.

## What you will learn

- The full lifecycle of a Decision, from raise to resolution to resumption
- How to configure Ollama and LM Studio as local LLM backends
- What Planning Sessions are and when to use them
- How to enable the local functionality feature flag

## Lessons in this module

1. The Decision Lifecycle in Depth
2. Setting Up Local LLM Backends
3. Planning Sessions
4. Module Quiz

> [!NOTE]
> Some features in this module require the **Local Functionality** feature flag to be enabled. This is covered in the lesson on local LLM backends.

## Key Takeaway

Multi-agent workflows reach their full potential when decisions flow smoothly, local models are available for private workloads, and planning sessions decompose complex goals before execution begins.
