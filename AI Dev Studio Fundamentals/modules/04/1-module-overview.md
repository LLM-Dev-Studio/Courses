# Module 4 Overview: Working with Agents

Agents are the workers in AI Dev Studio. This module shows you how to configure them, give them work via the Kanban board, and understand the inbox-and-message flow that connects the board to agent sessions.

## What you will learn

- The built-in agent roles and what each one is designed for
- How to start, stop, and monitor an agent session
- How the Kanban board works and how to assign tasks to agents
- How Messages flow from the board into an agent's inbox

## Lessons in this module

1. Agent Roles and Specialisation
2. Running and Monitoring Agent Sessions
3. The Kanban Board
4. The Inbox and Message Flow
5. Module Quiz

> [!NOTE]
> This module uses the VS Code extension panels introduced in Module 3. If you have not completed Module 3 yet, do so before continuing.

## Key Takeaway

Effective use of AI Dev Studio comes down to giving agents well-scoped tasks on the board and staying responsive to the Decisions and Messages they generate.
