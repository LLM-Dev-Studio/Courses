# Welcome to AI Dev Studio Fundamentals

AI Dev Studio is a multi-agent AI orchestration platform that coordinates specialised AI agents — Developer, Architect, QA, DevOps, and more — to autonomously execute software development tasks on your local machine.

This course walks you through everything you need to go from zero to productive: understanding the core concepts, setting up the platform, using the VS Code extension for day-to-day development, and leveraging the WinUI desktop app for deeper project management.

## What you will learn

By the end of this course you will be able to:

- Explain the purpose and architecture of AI Dev Studio
- Set up and run the platform for the first time
- Use the VS Code extension sidebar panels to monitor and interact with agents
- Create tasks, assign them to agents, and resolve decisions
- Navigate and use the WinUI desktop app for full project management
- Configure multi-agent workflows with local and cloud LLM backends
