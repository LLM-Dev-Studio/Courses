# Quiz Behavior and Progress

Quiz flow should be obvious and forgiving.

## Expected Experience

- Learner chooses an answer
- Learner clicks Next Question (or Continue on the final question)
- Wrong answers get visual feedback
- After max attempts, correct option is revealed and learner can proceed

> [!NOTE]
> This is a learning aid, not a strict high-stakes exam.

## Key Takeaway

Feedback should guide, not block, learning.
