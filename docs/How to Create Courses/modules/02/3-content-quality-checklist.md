# Content Quality Checklist

Use this before release.

## Checklist

- [ ] All lessons have clear titles
- [ ] No duplicated IDs in `course.json`
- [ ] Quiz frontmatter validates
- [ ] External links are intentional and current
- [ ] Key takeaway exists for each lesson

## Quick QA Commands

```bash
npm run lint
npm run build
```
