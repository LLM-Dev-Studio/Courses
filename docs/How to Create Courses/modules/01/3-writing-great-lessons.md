# Writing Great Lessons

Use a repeatable structure so learners can scan quickly.

## Suggested Pattern

1. Context: Why this lesson matters
2. Core concept: The one thing to understand
3. Practical steps: What to do now
4. Recap: A clear takeaway

> [!SUCCESS]
> Lessons with a consistent structure are easier to update and localize.
