# Module 2 Overview: Getting Started

Knowing what the platform does is one thing — having it running is another. This module takes you from a fresh machine to a working AI Dev Studio installation.

## What you will learn

- The prerequisites needed before you install anything
- The three different ways to run AI Dev Studio and when to choose each
- How to run the platform for the first time and confirm it is working
- How workspace and project layout works on disk

## Lessons in this module

1. Prerequisites
2. Choosing Your Setup
3. Your First Run
4. Workspace and Project Layout
5. Module Quiz

> [!IMPORTANT]
> If you are on a Mac or Linux machine, the WinUI desktop app is not available. You can still use the VS Code extension and the Aspire web host. Module 5 (WinUI) will not apply to you.

## Key Takeaway

Getting started requires three things: a .NET 10 SDK, at least one LLM backend, and a decision about which UI surface you will use first.
