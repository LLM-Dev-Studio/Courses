# Module 3 Overview: The VS Code Extension

The VS Code extension is the most common way to interact with AI Dev Studio during daily development. It surfaces agent activity directly in your editor's sidebar — no context switching required.

## What you will learn

- How to install and configure the extension
- What each of the four sidebar panels shows and how to use it
- How the extension discovers projects and connects to the backend
- How to resolve Decisions without leaving VS Code

## Lessons in this module

1. Installation and Project Detection
2. The Four Sidebar Panels
3. How the Extension Connects to the Backend
4. Resolving Decisions from VS Code
5. Module Quiz

> [!TIP]
> Keep the Logs panel open when first learning the extension — it shows exactly what the extension is doing at each step, including backend startup and SignalR connection events.

## Key Takeaway

The VS Code extension puts four purpose-built panels in your sidebar (Agents, Messages, Decisions, Logs) so you can monitor and direct AI agents without leaving your editor.
